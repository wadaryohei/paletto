## Palettoとは？
人の感情に寄り添ったプロダクトをつくる。
そんな「エモーショナルなプロダクト開発」をすることが私達「Paletto.」のVisionです。

感情を揺さぶるプロダクトでこそ、人の心は動きます。
「Paletto.」では課題解決のその先の「プロダクト」の思いをデザイン・開発でカタチにします。
